# Microservicios

Proyecto de microservicios para e-commerce librería.

## Stack
- Java 21, Spring Boot 3.3.1, Spring Cloud 2023.0.2
- PostgreSQL, Flyway, JWT, Feign, Eureka
- Docker Compose

## Servicios
- config-server, discovery-server, api-gateway
- user, product, inventory, cart, order
- analytics-dashboard (Streamlit)
